ReadMe file for EXAMPLE.ZIP

Most Zip files have a README.TXT, or
similar file to help you get started
with the file you have downloaded.
It might be named README.1ST,
READTHIS.TXT, INSTALL.TXT, etc.  In
any case, it's always a good idea to
look it over before taking any
further action.

WinZip makes it easy for you to open
files such as this before you unzip
the file.

TIP:  It's not always a good idea to
try to open just any file; some are
support files not meant to be opened
or may need other files in the zip
to operate properly.
